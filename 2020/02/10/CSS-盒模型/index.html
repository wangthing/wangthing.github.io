##IE模型和标准模型

标准模型：宽度和高度是content的大小,设置方法 box-sizing:content-box

IE模型： 宽高是content + padding + boeder，也就是我们说的 boeder-box

JS如何获取盒模型对应的高和宽

  • dom.style.width/height

    但是这样只能取得内联样式

  • dom.currentStyle.width/height

    可以获取所有样式计算之后的属性，但是只能在IE

  • window.getComputedStyle(dom).width/heiht

    兼容性更好，火狐谷歌等浏览器都可以

  • dom.getBoundingClientReact().width/height

    可以获取到元素局浏览器视口的绝对位置
